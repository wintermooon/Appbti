## **`🔌**GIT MERGE`

---

- git merge 하기
    1. git checkout dev
    2. git merge origin/피처브랜치
    -> merge conflict 생기는 부분은 내용 수정한 후 저장 -> add, commit, push
- merge한 내용 각자 피처 브랜치로 가져오기
    1. git checkout 피처 브랜치
    2. git pull origin dev
